# SensorsProject

##  תיאור כללי
משחק חקירה לוגית שבו השחקן מנסה לחשוף את "התמהיל הסודי" של סנסורים שמגלים כל סוכן איראני. כל סוכן דורש מספר סנסורים תואם כדי להיחשף. השחקן לא יודע את התמהיל מראש ומשתמש בניסוי וטעיה כדי לחשוף אותו.

## קלאסים בפרוייקט

- `Agent.cs`
שם:
דרגה: זוטר, מ"ב, מ"פ, בכיר
חולשה לסנסורים: 2 סנסורים שנקבעים אקראית 
(רשימת השוואות סנסורים)

מתודת activete() שבודקת אם הסנסורים שרוצים להצמיד לסוכן נמצאים ברשימת החולשות שלו

- `Sensor.cs`
שם:
סוג: טרמי / קלאסי / סלולרי ועוד

- `GameManager.cs` 
- אחראי על ניהול תורות חקירה וניהןל המשחק.


## זרימת המשחק
1. בכל תור השחקן מצמיד סנסורים לסוכן.
2. המערכת מחזירה כמה מהסנסורים שהוצמדו תואמים לתמהיל הסודי.
3. כאשר כל הסנסורים הנכונים מוצמדים – הסוכן נחשף.
